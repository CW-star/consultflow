# ConsultFlow

A full-stack practice management system built for solo consultants to eliminate 
payment tracking gaps, automate invoicing, and reclaim billable hours lost to 
manual administration.

## The Problem It Solves
- 48% payment visibility gap (invoices with unknown status)
- 14-day delays between completing work and sending invoices
- 6-8 hours/week lost to manual spreadsheet management
- Zero automated follow-up on overdue invoices

## Features
- Session logging with automatic charge calculation
- One-click invoice generation with PDF export
- Automated payment reminder sequences (3, 7, 14, 30 days)
- Real-time payment status dashboard
- Multi-client, multi-currency support (CAD/USD)
- Client payment history and analytics

## Tech Stack
- **Frontend**: Next.js 16, TypeScript, Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Auth, Storage, Edge Functions)
- **Email**: Resend
- **PDF**: React-PDF
- **Deployment**: Vercel

## Built For
Solo consultants and small consulting firms managing multiple clients 
and hourly billing across different rate tiers.
